# SmartLeadGen 🚀

An automated Python bot that extracts highly targeted local B2B leads from Google Maps and websites, directly syncing contact details and emails to Google Sheets in real-time. 

Perfect for digital marketing agencies, consultants, and freelancers looking to automate their client acquisition process.

## ✨ Features
* **Dynamic Search:** Input any Keyword (e.g., *Fashion Showroom, Real Estate*) and Location (e.g., *Tangail, Dhaka, Bangladesh*) directly in the terminal.
* **Maps Scraping:** Automatically scrolls and extracts Business Name, Phone Number, Address, and Website URL from Google Maps.
* **Deep Email Crawling:** Visits extracted websites and scans the homepage and "Contact Us" pages to hunt down email addresses using Regex.
* **Live Google Sheets Sync:** Connects to Google Sheets API to append newly found leads instantly.
* **Anti-Blocking:** Uses realistic User-Agents and intelligent delays to mimic human behavior.

## 🛠️ Tech Stack
* **Language:** Python 3.x
* **Automation:** Playwright / Selenium
* **Scraping:** BeautifulSoup4, Regex
* **Database:** Google Sheets API (`gspread`)
